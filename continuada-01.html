  2. Considere a ordenação de \(n\) números armazenados no vetor \(A = \langle a_1, a_2, \ldots, a_n \rangle\), localizando primeiro o menor elemento de \(A\) e permutando esse elemento com o elemento contido em \(A[1]\). Em seguida, determine o segundo menor elemento  de \(A\) e permute-o com \(A[2]\). Continue dessa maneira para os primeiros \(n-1\) elementos de \(A\). Escreva o pseudocódigo para esse algoritmo, conhecido como ordenação por seleção. Forneça os tempos de execução \(T(n)\) do melhor caso e do pior caso da ordenação por seleção. O cálculo de \(T(n)\) deve ser feito de modo formal, como foi realizado em sala para a ordenação por inserção. 

    
        selectionSort(A, n)
          para indice = 0 até n-1 faça
              indiceMenor = indice;
              para indiceSeguinte = indice + 1 até n faça
                se A[indiceSeguinte] < A[indiceMenor] então
                  indiceMenor = indiceSeguinte;

              aux = A[indice];
              A[indice] = A[indiceMenor];
              A[indiceMenor] = aux;

    O melhor caso ocorre quando o vetor já está ordenado, ou seja, não é necessário realizar nenhuma troca. Nesse caso, o tempo de execução é \(T(n) = n-1\).

    O pior caso ocorre quando o vetor está ordenado de forma decrescente, ou seja, é necessário realizar \(n-1\) trocas. Nesse caso, o tempo de execução é \(T(n) = \dfrac{n^2-n}{2}\).

  3. Prove as seguintes afirmações:

    • \( (n+1)^2 = O(n^2) \)

      A notação \(O(n)\) nos diz que existem constantes positivas \(c\) e \(n_0\) de modo que

      \(0 \le f(n) \le c * g(n) \text{ para todo } n \ge n_0\)

      Ademais, a partir da notação O, temos que

      \(f(n)=(n+1)^2\)
      \(g(n) = n^2\)

      Sendo assim, devemos provar que

      \((n+1)^2 \le c * n^2\)

      Pela propriedade dos produtos notáveis, podemos obter

      \((n+1)^2 = n^2 + 2n + 1\)

      Note que

      \(2n \le 2n^2 \text{ para todo } n \ge 0\)
      \(1 \le n^2 \text{ para todo } n \ge 1\)

      Logo

      \(n^2 + 2n + 1 \le n^2 + 2n^2 + n^2 = 4n^2 \text{ para todo } n \ge 1\)

      Portanto,

      \(f(n) \le c * g(n) \text{ para } c=4 \text{ e } n \ge n_0 = 1\)

    • \( n^2-3n-18 = \Omega(n^2) \)

      A notação \(\Omega(n)\) nos diz que existem constantes positivas \(c\) e \(n_0\) de modo que

      \(0 \le c * g(n) \le f(n) \text{ para todo } n \ge n_0\)

      Ademais, a partir da notação \(\Omega\), temos que

      \(f(n)=n^2 -3n -18\)
      \(g(n) = n^2\)

      Sendo assim, devemos provar que

      \(n^2 -3n -18 \ge c * n^2\)

      Na sequência, podemos dividir toda a inequação por \(n^2\), a fim de de isolar a constante \(c\), obtendo

      \begin{equation} \dfrac{n^2}{n^2} - \dfrac{3n}{n^2} - \dfrac{18}{n^2} \ge \dfrac{c * n^2}{n^2} \end{equation}
      \begin{equation} \dfrac{\cancel{n^2}}{\cancel{n^2}} - \dfrac{3 * \cancel{n}}{n*\cancel{n}} - \dfrac{18}{n^2} \ge \dfrac{c * \cancel{n^2}}{\cancel{n^2}} \end{equation}
      \begin{equation} 1 - \dfrac{3}{n} - \dfrac{18}{n^2} \ge c \end{equation}

      Supondo \(n_0 = 10\) e substituindo \(n\) por \(n_0\) na inequação, podemos obter

      \begin{equation} 1 - \dfrac{3}{10} - \dfrac{18}{10^2} \ge c \end{equation}
      \begin{equation} 1 - 0,3 - \dfrac{18}{100} \ge c \end{equation}
      \begin{equation} 1 - 0,3 - 0,18 \ge c \end{equation}
      \begin{equation} 1 - 0,48 \ge c \end{equation}
      \begin{equation} 0,52 \ge c \end{equation}

      Portanto,

      \(f(n) \ge c * g(n) \text{ para } c=0,52 \text{ e } n \ge n_0 = 10\)

    • \( n^3-3n^2-n = \Omega(n^3) \)

      A notação \(\Omega(n)\) nos diz que existem constantes positivas \(c\) e \(n_0\) de modo que

      \(0 \le c * g(n) \le f(n) \text{ para todo } n \ge n_0\)

      Ademais, a partir da notação \(\Omega\), temos que

      \(f(n)=n^3-3n^2-n\)
      \(g(n) = n^3\)

      Sendo assim, devemos provar que

      \(n^3-3n^2-n \ge c * n^3\)

      Na sequência, podemos dividir toda a inequação por \(n^3\), a fim de de isolar a constante \(c\), obtendo

      \begin{equation} \dfrac{n^3}{n^3} - \dfrac{3n^2}{n^3} - \dfrac{n}{n^3} \ge \dfrac{c * n^3}{n^3} \end{equation}
      \begin{equation} \dfrac{\cancel{n^3}}{\cancel{n^3}} - \dfrac{3 * \cancel{n^2}}{n*\cancel{n^2}} - \dfrac{\cancel{{n}}}{n^2*\cancel{n}} \ge \dfrac{c * \cancel{n^3}}{\cancel{n^3}} \end{equation}
      \begin{equation} 1 - \dfrac{3}{n} - \dfrac{1}{n^2} \ge c \end{equation}

      Supondo \(n_0 = 10\) e substituindo \(n\) por \(n_0\) na inequação, podemos obter

      \begin{equation} 1 - \dfrac{3}{10} - \dfrac{1}{10^2} \ge c \end{equation}
      \begin{equation} 1 - 0,3 - \dfrac{1}{100} \ge c \end{equation}
      \begin{equation} 1 - 0,3 - 0,01 \ge c \end{equation}
      \begin{equation} 1 - 0,31 \ge c \end{equation}
      \begin{equation} 0,69 \ge c \end{equation}

      Portanto,

      \(f(n) \ge c * g(n) \text{ para } c=0,69 \text{ e } n \ge n_0 = 10\)

    • \(\lg(100n^3+200n+300)^2 = O(\lg n)\)

      A notação \(O(n)\) nos diz que existem constantes positivas \(c\) e \(n_0\) de modo que

      \(0 \le f(n) \le c * g(n) \text{ para todo } n \ge n_0\)

      Ademais, a partir da notação O, temos que

      \(f(n)=lg(100n^3+200n+300)^2\)
      \(g(n) = lg(n)\)

      Sendo assim, devemos provar que

      \(lg(100n^3+200n+300)^2 \le c * lg(n)\)

      Pela propriedade do logaritmo de uma potência, podemos obter

      \(2 * lg(100n^3+200n+300) \le c * lg(n)\)

      Note que

      \(100n^3 \le 100n^3 \text{ para todo } n \ge 0\)
      \(200n \le 200n^3 \text{ para todo } n \ge 0\)
      \(300 \le 300n^3 \text{ para todo } n \ge 1\)

      Logo

      \(2 * lg(100n^3+200n+300) \le 2 * lg(100n^3+200n^3+300n^3) \text{ para todo } n \ge 1\)

      Trabalhando com as propriedades do logaritmo de uma soma, logaritmo de um produto e logaritmo de uma potência, podemos obter

      \(2 * lg(100n^3+200n^3+300n^3) = 2 * lg(600n^3) = 2 * lg(600) + 2 * lg(n^3) = 2 * lg(600) + 6 * lg(n)\)

      Logo, temos que

      \(2 * lg(100n^3+200n+300) \le 2 * lg(600) + 6 * lg(n) \text{ para todo } n \ge 1\)

      Supondo \(c = 8\) e \(n_0 = 1\), e substituindo \(n\) por \(n_0\) na inequação, podemos obter

      \(2 * lg(100*1^3+200*1+300) \le 2 * lg(600) + 6 * lg(1)\)
      \(2 * lg(100+200+300) \le 2 * lg(600) + 6 * 0\)
      \(2 * lg(600) \le 2 * lg(600)\)

      Portanto,

      \(f(n) \le c * g(n) \text{ para } c=8 \text{ e } n \ge n_0 = 1\)
